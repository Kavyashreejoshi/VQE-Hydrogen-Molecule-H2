# VQE for H₂ Molecule ⚛️

Variational Quantum Eigensolver implementation to compute the ground-state energy of the Hydrogen molecule (H₂) at bond distance 0.74 Å using STO-3G minimal basis.

**Result**: **-1.13618 Hartree** (matches exact literature value within chemical accuracy)

### Features
- 4 qubits, Jordan-Wigner mapping
- Hardware-efficient ansatz
- Classical optimization (COBYLA)
- No Qiskit/Pennylane needed – runs with just NumPy + SciPy
- Fully runnable Jupyter notebook
